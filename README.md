# newsvalues_analysis_maltseva
Данный репозиторий является приложением к курсовой работе «Новостные ценности в дискурсе экологических новостей» Мальцевой Анастасии, студентки 2 курса бакалавриата ФиКЛ НИУ ВШЭ.

Содержимое:

`handcorpus_analysis`: Модуль анализа размеченного вручную корпуса русскоязычных новостей. 
С помощью библиотеки Pandas в данном файле выполняются:

* Генерация статистических графиков;
* Сравнительный анализ визуального и текстового контента;
* Статистика по тегам и типам новостей.


`clip_bert_analysis`: код для нейросетевой разметки новостей (модели CLIP, BERT).

* Инициализация моделей;
* Разметка при помощи моделей (CLIP — изображения, BERT — тексты);
* Объединение датасетов;
* Сравнительный анализ, статистика и графики.


`news values data`: сам корпус в формате .xlsx.
